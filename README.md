# Portfolio Website
Written in nuxt 4

## Nuxt Minimal Starter

### Setup
Install dependencies:
```bash
npm install
# or pnpm install | yarn install | bun install
````

### Development

Runs the dev server at `http://localhost:3000`:

```bash
npm run dev
# or pnpm dev | yarn dev | bun run dev
```

### Production

Build and preview:

```bash
npm run build
npm run preview
# or pnpm / yarn / bun equivalents
```
